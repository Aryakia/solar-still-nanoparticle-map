# Nanoparticle-Assisted Solar Still Study Map

This repository hosts an interactive web map of experimental studies on nanoparticle-assisted solar still desalination.

The map visualizes reviewed studies by reported experimental location, nanoparticle application category, and study details. It includes filters, search functionality, reference links, and downloadable dataset options.

## Website

The website is published using GitHub Pages:

https://aryakia.github.io/solar-still-nanoparticle-map/

## Features

- Interactive world map of reviewed solar still studies
- Study markers grouped by location and nanoparticle application category
- Search by country, location, nanoparticle, reference, and solar still type
- Filters for:
  - Basin nanofluid
  - External-circuit nanofluid
  - Nano-coating
  - Nano-enhanced PCM
- Reference links for each study
- Downloadable CSV dataset
- Downloadable Excel dataset
- Downloadable static SVG map

## Files

- `index.html` — main website file
- `solar_still_np_mapping_dataset_enhanced.xlsx` — Excel dataset
- `solar_still_np_location_map_static_world_english_v2.svg` — static map figure

## Method Note

Mapping uses reported experimental locations where available. Country-level coordinates are used when only a country was reported. Reference numbers open DOI links when available; otherwise, they open Google Scholar searches.
